Система управления общежитием с использованием PHP и MySQL
Шаги установки (настройка)

Скачайте и разархивируйте файл на вашем локальном компьютере.
Поместите этот файл в папку xampp/htdocs/.
Настройка базы данных
Откройте phpMyAdmin.
Создайте базу данных с именем "hostel".
Импортируйте базу данных из файла "hostel.sql".
Откройте ваш браузер и введите в адресной строке "http://localhost/hostel/".
Данные для входа
Чтобы войти в качестве администратора, введите в адресной строке "http://localhost/hostel".
